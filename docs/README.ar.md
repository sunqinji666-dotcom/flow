# Flow لنظام macOS

> اتصال بسيط، وحالة واضحة، وإعدادات محلية.

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **العربية**

إعداد وتواصل: **Jacksun (孙秦吉)** · [qinji@jack-sun.com](mailto:qinji@jack-sun.com)

Flow عميل اتصال خفيف لنظام macOS. يعمل المحرك والتحقق من العقد والتوجيه ووكيل النظام في الخلفية، بينما يعرض التطبيق حالة الاتصال والوكيل بوضوح.

لا يحتوي المصدر العام على خوادم حقيقية أو روابط اشتراك أو UUID أو بيانات اعتماد. احتفظ بإعداداتك الخاصة على جهاز Mac فقط. يتطلب macOS 14+ وApple Silicon. نزّل من [Releases](../../releases/latest) أو اقرأ [الدليل الصيني الكامل](README.zh-CN.md).
