# Flow for macOS

> 一鍵連線，狀態清楚，設定留在本機。

[简体中文](../README.md) · **繁體中文** · [English](README.en.md) · [日本語](README.ja.md)

Flow 是 macOS 的輕量連線用戶端。它將核心、節點驗證、路由和系統代理放在背景，只呈現是否連線、流量經過哪裡與系統代理狀態。

公開原始碼不含真實伺服器、訂閱網址、UUID 或任何憑據；請只在自己的本機保存私有設定。需要 macOS 14+ 與 Apple Silicon。可由 [Releases](../../releases/latest) 下載，或閱讀 [中文完整說明](README.zh-CN.md)。
