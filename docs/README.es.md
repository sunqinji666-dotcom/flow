# Flow for macOS

> Una conexión simple, estado claro y configuración local.

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **Español**

Flow es un cliente de conexión ligero para macOS. Mantiene el núcleo, la validación de nodos, las rutas y el proxy del sistema en segundo plano, y muestra claramente el estado de conexión y del proxy.

El código público no incluye servidores reales, URL de suscripción, UUID ni credenciales. Conserva tu configuración privada solo en tu Mac. Requiere macOS 14+ y Apple Silicon. Descarga desde [Releases](../../releases/latest) o consulta la [guía completa en chino](README.zh-CN.md).
