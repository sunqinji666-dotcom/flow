# Flow для macOS

> Простое подключение, понятный статус и локальная конфигурация.

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **Русский**

Flow — лёгкий клиент подключений для macOS. Ядро, проверка узлов, маршрутизация и системный прокси работают в фоне, а приложение ясно показывает состояние подключения и прокси.

В открытом коде нет реальных серверов, URL подписок, UUID или учётных данных. Храните личные настройки только на своём Mac. Требуются macOS 14+ и Apple Silicon. Скачайте через [Releases](../../releases/latest) или прочитайте [полное руководство на китайском](README.zh-CN.md).
