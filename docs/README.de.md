# Flow for macOS

> Ein Klick zur Verbindung, klarer Status, lokale Konfiguration.

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **Deutsch**

Flow ist ein schlanker Verbindungs-Client für macOS. Kern, Knotenprüfung, Routing und Systemproxy arbeiten im Hintergrund; sichtbar bleiben Verbindungs- und Proxy-Status.

Der öffentliche Quellcode enthält keine echten Server, Abonnement-URLs, UUIDs oder Zugangsdaten. Bewahre private Einstellungen ausschließlich auf deinem Mac auf. Erfordert macOS 14+ und Apple Silicon. Download über [Releases](../../releases/latest) oder die [vollständige chinesische Anleitung](README.zh-CN.md).
