# Flow for macOS

> एक क्लिक में कनेक्शन, स्पष्ट स्थिति और स्थानीय कॉन्फ़िगरेशन।

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **हिन्दी**

निर्माता व संपर्क: **Jacksun (孙秦吉)** · [qinji@jack-sun.com](mailto:qinji@jack-sun.com)

Flow macOS के लिए हल्का कनेक्शन क्लाइंट है। इसका कोर, नोड सत्यापन, रूटिंग और सिस्टम प्रॉक्सी पृष्ठभूमि में रहते हैं; ऐप कनेक्शन और प्रॉक्सी की स्थिति स्पष्ट रूप से दिखाता है।

सार्वजनिक स्रोत में वास्तविक सर्वर, सदस्यता URL, UUID या क्रेडेंशियल शामिल नहीं हैं। अपनी निजी सेटिंग केवल अपने Mac पर रखें। macOS 14+ और Apple Silicon आवश्यक हैं। [Releases](../../releases/latest) से डाउनलोड करें या [चीनी विस्तृत गाइड](README.zh-CN.md) पढ़ें।
