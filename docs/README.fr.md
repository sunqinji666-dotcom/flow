# Flow for macOS

> Une connexion simple, un état clair et une configuration locale.

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **Français**

Créé par **Jacksun (孙秦吉)** · [qinji@jack-sun.com](mailto:qinji@jack-sun.com)

Flow est un client de connexion léger pour macOS. Il garde le moteur, la validation des nœuds, le routage et le proxy système en arrière-plan, tout en affichant clairement l’état de la connexion et du proxy.

Le code public n’inclut ni serveur réel, ni URL d’abonnement, ni UUID, ni identifiant. Gardez vos réglages privés uniquement sur votre Mac. macOS 14+ et Apple Silicon sont requis. Téléchargez depuis [Releases](../../releases/latest) ou consultez le [guide chinois complet](README.zh-CN.md).
