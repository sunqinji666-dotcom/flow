# Flow for macOS

> 한 번의 연결, 분명한 상태, 설정은 로컬에만.

[简体中文](../README.md) · [繁體中文](README.zh-TW.md) · [English](README.en.md) · [日本語](README.ja.md) · **한국어**

제작 및 문의: **Jacksun (孙秦吉)** · [qinji@jack-sun.com](mailto:qinji@jack-sun.com)

Flow는 macOS용 경량 연결 클라이언트입니다. 코어, 노드 검증, 라우팅, 시스템 프록시는 뒤에서 처리하고 연결 상태와 시스템 프록시 상태만 명확하게 보여 줍니다.

공개 소스에는 실제 서버, 구독 URL, UUID, 자격 증명이 포함되지 않습니다. 개인 설정은 반드시 자신의 Mac에만 보관하세요. macOS 14+ 및 Apple Silicon이 필요합니다. [Releases](../../releases/latest)에서 내려받거나 [중국어 전체 안내](README.zh-CN.md)를 참고하세요.
