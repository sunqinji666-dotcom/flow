# Flow untuk macOS

> Satu klik untuk terhubung, status jelas, konfigurasi tetap lokal.

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **Bahasa Indonesia**

Dibuat oleh **Jacksun (孙秦吉)** · [qinji@jack-sun.com](mailto:qinji@jack-sun.com)

Flow adalah klien koneksi ringan untuk macOS. Mesin inti, validasi node, perutean, dan proksi sistem berjalan di latar belakang; aplikasi menampilkan status koneksi dan proksi secara jelas.

Kode sumber publik tidak berisi server asli, URL langganan, UUID, atau kredensial. Simpan konfigurasi pribadi hanya di Mac Anda. Memerlukan macOS 14+ dan Apple Silicon. Unduh dari [Releases](../../releases/latest) atau baca [panduan lengkap berbahasa Tionghoa](README.zh-CN.md).
