# Flow for macOS

> Conexão simples, status claro e configuração local.

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **Português (Brasil)**

Flow é um cliente de conexão leve para macOS. Ele mantém o núcleo, a validação de nós, o roteamento e o proxy do sistema em segundo plano, mostrando com clareza o estado da conexão e do proxy.

O código público não contém servidores reais, URLs de assinatura, UUIDs ou credenciais. Guarde suas configurações privadas somente no seu Mac. Requer macOS 14+ e Apple Silicon. Baixe em [Releases](../../releases/latest) ou leia o [guia completo em chinês](README.zh-CN.md).
